# Manuscript Contributor Sandbox

Feel free to put whatever code files you want in this folder! A good rule of thumb is to just name the file with your name so that no one accidentally messes with your stuff (or vice versa).

Also, please _do not_ add any code inputs/outputs (e.g., data, graphs) to the GitHub repo. Please, upload those to the Drive or share them via email/etc. to share with the group!

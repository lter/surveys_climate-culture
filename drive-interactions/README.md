# Google Drive Interactions

The scripts in this folder interact with a particular subfolder of the LTER Network Office's Shared Google Drive. **For the code in this folder to be run-able, you need (1) access to that folder, and (2) to have authenicated with the `googledrive` R package.**

For help with the latter point, see this [handy tutorial](https://lter.github.io/scicomp/tutorial_googledrive-pkg.html), developed by the LTER Scientific Computing team!
